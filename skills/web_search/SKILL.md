# web_search

一句话简介：通过联网搜索/打开网页/PDF 截图获取最新信息，并输出带来源引用的结论。

## When to use
- 需要最新/可能变化的信息（新闻、政策、版本、价格、规格等）
- 需要给出可核验的来源引用
- 需要读取网页或 PDF（含表格/图片）

## What it does
- 关键词搜索（多来源交叉验证）
- 打开结果页并定位关键段落
- PDF 按页截图读取表格/图
- 汇总为结构化要点，并在关键结论后附引用

## Playbook
1. search：用 2–4 组 query 找 3–6 个高质量来源（官方/一手文档优先）。
2. open：打开最相关来源；必要时 find 定位关键词。
3. PDF：必须逐页 screenshot 阅读表格/图片（不要只依赖文本解析）。
4. 输出：结论 + 证据（每段末尾附引用）+ 不确定性/冲突来源说明。
